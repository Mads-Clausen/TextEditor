Welcome to the official site of the text editor "Da VIMci"!
This is a work-in-progress project, which will contain the following features, at least:

  • Customisable Syntax Highlighting
  • A simple API for plugins(to allow for custom views, etc.)
  • Rich text editing(multiple cursors, stuff like that)
  • Sleek and simple user interface
  • Console for realtime scripting
This section contains images of the current state of the editor. For more images, go here.


There is currently no real build, but it is pretty easy to compile the editor.
You can get the source at the Github page.
Alot of information about compiling the project can be found in the README.md file.
I strongly encourage you to report any issues that you find in the editor.
This is a very important process of developing an application, as you want to be certain
that the users can enjoy it. Before reporting an issue, make sure to read the guidelines.

Guidelines for reporting issues:
  • Check if it has already been reported or closed.
  • Make sure that you are using the newest version of the application.
  • Provide information about the issue, like crash screens, error messages etc.
  • Be sure to mark your report with the correct label(s).
